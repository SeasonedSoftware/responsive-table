# ResponsiveTable

## Install

```bash
yarn add @seasoned-cc/responsive-table react react-dom @material-ui/core lodash
```

## Usage

```jsx
import React from 'react'

import { ResponsiveTable } from '@seasoned-cc/responsive-table'

export default () => (
  <ResponsiveTable />
)
```
